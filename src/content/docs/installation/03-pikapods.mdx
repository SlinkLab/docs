[PikaPods](https://www.pikapods.com/) is a managed hosting platform that makes it easy to run open-source applications with minimal setup. Deploy Slink on PikaPods with just one click and let them handle the infrastructure.

## Quick Deployment

Click the button below to deploy Slink on PikaPods:

[![Run on PikaPods](https://www.pikapods.com/static/run-button.svg)](https://www.pikapods.com/pods?run=slink)

## Configuration

<Aside type="caution" title="Important">
  Set the admin user environment variables during deployment so Slink creates
  your first admin account automatically — the easiest way to get started.
</Aside>

In the PikaPods environment configuration, set `ADMIN_EMAIL` and `ADMIN_PASSWORD` (and optionally `ADMIN_USERNAME`) before launching. On first startup, Slink creates and activates the admin account for you.

See [First User Setup](/getting-started/03-first-user-setup/) for how automatic admin creation works, and the [Environment Variables](/configuration/01-environment-variables/) reference for all available options.

## After Deployment

<Steps>

1. Once deployment is complete, PikaPods will provide you with a URL to access your Slink instance.

2. Navigate to your instance URL and log in with the admin credentials you configured.

3. Start uploading and sharing images!

4. Customize your instance settings through the admin panel or by updating environment variables in the PikaPods dashboard.

</Steps>
