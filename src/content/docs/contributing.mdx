Contributions of all kinds are welcome. For major changes, please open an [issue](https://github.com/andrii-kryvoviaz/slink/issues) first to discuss what you would like to change.

## Ways to Contribute

- **Code**: open a [pull request](https://github.com/andrii-kryvoviaz/slink/pulls) with new features, bug fixes, or improvements.
- **Bugs and features**: report them through [GitHub issues](https://github.com/andrii-kryvoviaz/slink/issues).
- **Translations**: help improve or add a language. See [Localization](/usage/05-localization/).
- **Documentation**: fixes and additions are always appreciated.

## Development Setup

Fork the [repository](https://github.com/andrii-kryvoviaz/slink) and start a development environment with `make run-dev`. See the [README](https://github.com/andrii-kryvoviaz/slink#readme) for prerequisites.

## Security Issues

<Aside type="caution">
  Report security vulnerabilities privately by email as described in the [Security Policy](https://github.com/andrii-kryvoviaz/slink/blob/main/SECURITY.md). Please do not open a public GitHub issue for security problems.
</Aside>

## Code of Conduct

This project follows a [Code of Conduct](https://github.com/andrii-kryvoviaz/slink/blob/main/CODE_OF_CONDUCT.md). By participating, you are expected to uphold it.
