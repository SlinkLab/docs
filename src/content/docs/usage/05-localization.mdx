Slink's interface is available in nine languages, with English as the source and eight translations included.

## Supported Languages

| Code | Language    |
| ---- | ----------- |
| `en` | English (source) |
| `de` | Deutsch     |
| `es` | Español     |
| `fr` | Français    |
| `it` | Italiano    |
| `ja` | 日本語      |
| `pl` | Polski      |
| `uk` | Українська  |
| `zh` | 中文        |

## Changing Your Language

Go to your **Preferences** page (`/preferences`) and pick a language under **Display Language**. The change applies immediately and is persisted with your account, so it follows you across devices and browsers.

Anonymous visitors fall back to English unless their session cookie already carries a previously selected locale.

<Aside type="note">
  Most of the translations in this release are AI-generated, so wording may
  feel off in places. Contributions to improve any language are very welcome.
  See the [Contributing](/contributing) guide.
</Aside>
